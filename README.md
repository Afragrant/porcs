# PROCS — Pantograph-Rigid Overhead Contact System

基于[受电弓与刚性接触网动力相互作用研究](./受电弓与刚性接触网动力相互作用研究_关金发.pdf)的假设模态法，实现受电弓-刚性接触网系统动力学仿真。

## 模型概览

| 模块 | 建模方法 |
|------|----------|
| 刚性接触网 | Euler-Bernoulli 简支梁，假设模态法（`NM` 阶模态截断） |
| 受电弓 | 三质量块归算模型（弓头-中间-底架） |
| 弓网接触 | 线性弹簧 + 单边约束（`KS`），active-set 迭代求解 |
| 瑞利阻尼 | `C = α·M + β·K` |
| 波磨不平顺 | 余弦波模型 $W_{cw} = \frac{1}{2}A_w[1 - \cos(2\pi l / \lambda_w)]$ |

### 数值方法

- **时间积分**：Newmark-β 法（β=0.25, γ=0.5，平均加速度法）
- **接触迭代**：active-set 策略（<5 次收敛），利用 Woodbury / Sherman-Morrison 公式将每步 O(n³) 降为 O(n²)
- **系统矩阵**：P = K + a₀M + a₁C 做一次 Cholesky 分解，全时程复用

## 文件结构

```
procs/
├── procs.py           # 核心仿真模块（动力学模型 + 求解器 + 绘图）
├── procs_to_data.py   # 运行仿真并导出稳定段接触力 CSV
├── pyproject.toml
└── result/
    ├── pc_plots/      # 静力形态图 + 弓网响应图
    └── pc_data/       # 稳定段接触力数据 (x[m], contact_force[N])
```

## 快速开始

```bash
uv run python procs.py          # 运行仿真 + 绘图
uv run python procs_to_data.py  # 运行仿真 + 导出 CSV
```

## 输出

### 接触力统计量（稳定段）

输出均值、标准差、最大值、最小值、离线率（%），例如：

```
--- Contact force statistics (stable window: 前6跨–后6跨) ---
  mean_N                    : 123.456
  std_N                     : 23.456
  max_N                     : 234.567
  min_N                     : 0.000
  loss_of_contact_pct       : 0.123
```

### 图形

- `initial_catenary_shape_c{预设}.png` — 刚性接触网初始静力形态（含悬挂点/中间接头标注）
- `pc_response_p{受电弓}_c{接触网}_{速度}kmh.png` — 3×2 子图：全部段 / 稳定段的接触力、弓头位移、接触网位移
